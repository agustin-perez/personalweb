Sobre los servidores

______________

Actualmente cuento con:

2 servidores IBM System x3200 M2 con:
 Intel Xeon E3110
 6 GB de RAM DDR2 ECC - UDIMM
 Controladora RAID - SAS LSI Logic 1064E (mini pci)
 Hypervisor: VMware ESXi 6.5.0

A pesar de ser iguales, tienen diferentes configuraciones de almacenamiento:
  Servidor principal: RAID 1 de 2TB en conjunto con otro RAID 1 de 0.5TB
  Servidor secundario: RAID 0 de 0.5TB

Estos servidores se diferencian en:
 Servidor principal:
   - Web server
   - Base de Datos PostgreSQL
   - Servidores de juegos
   - Servidor local (NAS, impresión, entre otros)
   - Base de Datos Informix (No utilizada)

 Servidor secundario:
   - Backups del principal
   - Servicios de administración (Control de UPS, DDNS, entre otros)
   - Servidor Web auxiliar

Red:
 - Firewall pfSense corriendo sobre FreeBSD, con:
   - VPN
   - Portal cautivo
 - Router Ubiquiti EdgeRouter X
 - Red Gigabit en su totalidad
 - UPS offline de 1500VA
